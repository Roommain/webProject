02-第二章 Sublime text指南

一、 下载Sublime

下载地址：http://www.sublimetext.com/

二、 安装Sublime

三、 安装package control(插件管理器)

1 按下快捷键：ctrl+shift+p 打开命令行
2 然后在输入框中输入install，然后选中install package control
3 等待 安装，出现 success表示成功安装

四、安装插件

1 按下快捷键：ctrl+shift+p
2 输入Package Control: ，然后选中Install Package,进入安装界面
3 输入要安装的插件名称，如Chineselocalizations（汉化插件）

五 卸载插件

1 按下快捷键：ctrl+shift+p
2 输入Package Control: ，然后选中Remove Package,进入安装界面
3 输入要卸载的插件名称，如Chineselocalizations（汉化插件）

六 常用插件

Chinese 不用多说汉化插件
Color Highlighter 颜色高亮显示
Javascript Completions js代码提示
Emmet css选择器语法生成 eg: db 按下tab display:block
SublimeTmpl 模板插件 常用模板快捷键
ctrl+alt+h → html
ctrl+alt+j → js
ctrl+alt+c → css
AutoFilName 自动查找引入的文件路径
Color Picker 颜色选择器
让输入法显示在当前行下边 css选择器语法生成 eg: db 按下tab display:block
js代码格式化 css选择器语法生成 eg: db 按下tab display:block
IMESupport jsFormatCsscomb css格式化冲排列
LiveReload 实时预览 需要chrome浏览器同样安装此插件
SyncedSidebarBg 侧边栏背景颜色

七、配置文件

在首选项 -设置
在首选项 -快捷键设置 — 配置浏览器路径预览— 有预览还得安装SideBarEnhancements插件

八 常用快捷键操作

快捷键 说明
ctrl+shift+p 打开命令行
Alt+F3 选中文本按下快捷键，即可一次性选择全部的相同文本进行同时编辑
Ctrl+鼠标左键 可以同时选择要编辑的多处文本
ctrl+f 查找
ctrl+k 打开侧边栏(默认是ctrl+k+b)
ctrl+shift+F 在文件夹内查找，与普通编辑器不同的地方是sublime允许添加多个文件夹进行查找
Ctrl+D 选词 （反复按快捷键，即可继续向下同时选中下一个相同的文本进行同时编辑）
ctrl+/ 单行注释 // 再次使用快捷键解除注释
ctrl+shift+/ 多行注释/**/ 再次使用快捷键解除注释
Ctrl+Z 撤销
Ctrl+Y 恢复撤销
Ctrl+Shift+[ 折叠代码
Ctrl+Shift+] 展开代码
ctrl+shift+D 快速向下赋值当前行
ctrl+shift+↑ 当前行与上行交换位置
ctrl+shift+↓ 当前行与下行交换位置
shift+delete 删除当前一行
ctrl+j 合并一行
